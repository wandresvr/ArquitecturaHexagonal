# Poblar bases de datos

### Dar permisos:

```bash
sudo chmod +x init_database.sh
```

### Ejecutar:

```bash
sudo  ./backup_database.sh
```

#### Nota: 
Siempre ejecutar desde la carpeta ```database```.

# Realizar Backup

### Dar permisos:

```bash
sudo chmod +x backup_database.sh
```

### Ejecutar:

```bash
sudo  ./backup_database.sh
```
